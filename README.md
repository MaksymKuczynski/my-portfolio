## Як працює цей проєкт
Сайт побудований на HTML та CSS. Основна ідея — централізована сторінка з картками студентів, де кожна картка містить аватар та посилання на особисте портфоліо студента.

## Що я навчився
* Використовувати Git та GitHub для контролю версій та деплою (GitHub Pages).
* Створювати адаптивну верстку за допомогою **Flexbox**.
* Працювати з CSS-стилями (border-radius, box-shadow, object-fit для фото).
* Розв'язувати проблеми з кешуванням браузера та налаштуванням шляхів до файлів.
