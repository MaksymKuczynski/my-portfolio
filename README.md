## Опис проекту
Портфоліо розроблене для презентації моїх навчальних досягнень, виконаних лабораторних робіт та проектів. Основна увага приділена чистоті коду, адаптивності та сучасному візуальному стилю.

## Технології та навички
У процесі створення цього портфоліо я використав та закріпив наступні знання:
* **HTML5 & CSS3:** Побудова семантичної структури та стилізація елементів.
* **Flexbox:** Створення адаптивної сітки, яка коректно відображається на різних пристроях.
* **Git & GitHub:** Робота з гілками, виконання коммітів та керування Pull Request-ами.
* **Markdown:** Написання технічної документації для проекту.

## Кроки створення (Tutorial)
1.  **Проектування:** Визначення структури сторінки та вибір колірної гами (Deep Slate & Royal Blue).
2.  **Верстка:** Написання HTML-каркасу та стилізація через CSS із використанням `box-sizing: border-box` для усунення проблем з прокруткою.
3.  **Контроль версій:** Ініціалізація репозиторію, створення окремих гілок для нових функцій та фіксація змін.
4.  **Деплой:** Налаштування GitHub Pages для автоматичної публікації сайту.

## Як повторити цей проект
1. Склонуйте репозиторій: `git clone https://github.com/MaksymKuczynski/my-portfolio.git`
2. Відкрийте папку в Visual Studio Code.
3. Внесіть свої дані в `index.html` та замініть фото в папці `avatars`.
4. Запустіть файл через браузер для перегляду результату.
